# Plataforma HelpEnem
Projeto HelpEnem - Plataforma Educacional

A HelpEnem é uma plataforma de ensino online focada em ajudar estudantes de todas as idades a se prepararem para os principais vestibulares do país, com destaque para o Enem.
Oferecemos uma preparação completa por meio de videoaulas, correção de redações, simulados e diversas outras ferramentas que tornam o aprendizado mais eficiente, acessível e direcionado à aprovação.
